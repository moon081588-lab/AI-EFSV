8가지 통신 프로토콜, 각각 다른 규칙

BCM은 5개 프로토콜을 사용합니다. 각 프로토콜마다 verification 방식과 매핑 규칙이 다릅니다.

  • Timing 분석 필요

43 REQs
  • Schedule table 매핑
  • Frame ID 기준
  • Master/Slave 구분
FlexRay

  • Static/Dynamic 슬롯
  • Cycle 단위 검증
  • Cluster config 필요

  • SOME/IP 규격 준수
